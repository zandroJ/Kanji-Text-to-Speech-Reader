How It Works

This text-to-speech reader highlights each kanji character as it's pronounced. The system uses the Web Speech API for pronunciation and synchronizes the highlighting with the audio playback.

Features

  • Synchronized highlighting
  • Hover for reading/meaning
  • Progress indicator
  • Play/pause controls
  • Font selection

Instructions

  1. Click "Play All" to start
  2. Use "Pause" to stop temporarily
  3. Hover over any kanji for details
  4. Click Play again to resume
  5. Select different fonts using dropdown
